Practices

Starting with a mobile first design I create simple yet effective user experiences from mobile to desktop. Other practices include using rem’s instead of pixels for improved scalability, optimizing assets to the smallest possible file size to improve load times, scaling assets appropriately so they look crisp on all devices, as well as using the .svg format whenever possible.
